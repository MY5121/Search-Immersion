# Search Immersion

**这是一款能将Google搜索页面或新标签页转变为类似Google Nest Hub风格智能显示屏的扩展程序。**

当前为发布测试版（Beta版），因此部分环境下行为可能有所不同。敬请知悉。

---

## 特点

* **智能显示屏UI:** 采用毛玻璃卡片式设计。美观直观地显示天气、新闻、时钟。
* **动态媒体联动:**  当在YouTube或Spotify播放音乐时，背景会自动变为专辑封面，切换至沉浸感十足的音乐播放器模式。
* **Zen模式（专注模式）:** 一键隐藏时钟和卡片，仅保留悬浮的搜索栏，助您专注工作。
* **防烧屏功能:**  考虑到在显示屏上的常时显示需求，搭载了定期细微移动画面元素的保护功能。
* **可定制性:** 可通过设置界面详细调整背景亮度、玻璃透明度、时钟字体等。

---

## 安装方法

此扩展程序目前仅在“开发者模式”下可用，尚未通过Chrome网上应用店审核。

### 1. 文件下载与准备

1. 下载最新的源代码（ZIP文件）并解压。
2. **【重要】** 请使用解压后得到的 整个文件夹。

### 2. 在Chrome中加载的步骤

1. 打开您的浏览器（推荐Google Chrome）。
2. 在地址栏输入 **`chrome://extensions`**  并打开。
3. 打开页面右上角的 “开发者模式”。
4. 点击左上角的 “加载已解压的扩展程序” 按钮。
5. 选择步骤1中解压得到的 整个文件夹 并加载。

### 3. 使用方法

1. 访问Google首页。
2. Search Immersion的界面将自动显示。
3. 请通过屏幕中央下方停靠栏中的 **[⚙️] (設定)** 按钮，配置您偏好的设计和联动功能。
---

## 支持的设备

* 所有使用Chromium引擎的浏览器。
* 支持Mac、Windows、Linux。但由于在Mac上开发，在Windows环境下字体渲染和UI布局可能略有差异。
